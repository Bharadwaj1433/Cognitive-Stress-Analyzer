# 📖 Project Overview

**Cognitive Stress Analyzer using AI** is an intelligent web application designed to help users better understand their cognitive well-being through behavioral analysis and AI-powered insights.

Instead of functioning as a simple stress calculator, the application evaluates multiple lifestyle factors—including sleep, study habits, mood, screen time, physical activity, and daily routines—to estimate cognitive stress and provide meaningful recommendations.

The system integrates Google's Gemini AI to deliver personalized explanations, supportive therapist-style conversations, adaptive planning, and behavioral insights, creating a complete cognitive management platform.

# 🔑 Gemini API Setup

1. Open Google AI Studio.
2. Generate a Gemini API Key.
3. Locate your JavaScript configuration.
4. Replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

with your own API key.

> **Important:** Never expose your production API key in a public repository. For production deployments, use environment variables or a secure backend.

# 🌍 Deploying on Netlify

## Method 1 — Deploy from GitHub (Recommended)

1. Push your project to GitHub.
2. Log in to Netlify.
3. Click **Add new site** → **Import an existing project**.
4. Connect your GitHub account.
5. Select:

```
Cognitive-Stress-Analyzer
```

6. Build settings:

Build Command

```
(Leave Empty)
```

Publish Directory

```
/
```

7. Click **Deploy Site**.

Netlify will automatically redeploy your application whenever you push changes to GitHub.

---

## Method 2 — Drag & Drop

1. Open Netlify.
2. Choose **Deploy manually**.
3. Drag the project folder containing `index.html`.
4. Wait for deployment.
5. Netlify will generate a public URL.
